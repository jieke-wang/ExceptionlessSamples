# 构建Exceptionless
```shell
mkdir -p /home/exceptionless
cd /home/exceptionless
```

下载https://github.com/exceptionless/Exceptionless/blob/master/samples/docker-compose.yml到/home/exceptionless

```shell
docker-compose up -d
```

访问地址: http://<你的服务器IP>:5000/
自行注册一个账号

# 使用Exceptionless
